Mesmo eu tendo feito alguns cursos em Java, eu sempre preferi web em C#, eu comecei a programar em C#, gosto muito do Visual Studio e acho a melhor IDE para ajudar na programação.

Esse CRUD é feito em Web Forms usando o FrameWork NHibernate e suas bibliotecas. Há validações de campos, usando Javascript. É um CRUD completo com todas as funcionalidades, inclusive há uma conexão com o Banco de Dados SQL server local.

O propósito é praticar os conhecimentos adquiridos em meu Estágio de Programação -iniciei há dois meses-. Há muito o que se aprender ainda, mas já da para perceber o quanto vou melhorando as práticas de Programação com o passar do tempo.

IDE Usada: Microsoft Visual Studio 2015; DataBase: SQL Server; Tecnologia: Web Forms; FrameWork: NHibernate; HTML5 + CSS3 + Javascript; Consultas utilizando LINQ.
